Importar Dataframes

Existen varios formatos y varias maneras para trabajar con dataframes en pandas. Los formatos mas utilizados son archivos de excel, csv, dta. El metodo para importar dichos archivos cambiara dependiendo del tipo. Veamos algunos ejemplos: 

 
							# Importar
							"pd.read_excel('base_de_datos.xlsx')" # excel
							"pd.read_csv('datos.csv')" # csv
							"pd.read_stata('datos.dta')" #dta

Diferentes formas de mostrar el DataFrame

En pandas dependiendo de la necesidad que tengamos podemos mostrar de diferentes formas el dataframe; un resumen, primeros 5 datos, entre otras. 

 
						# metodos para hacerlo
						"head()" # 5 primeros datos.
						"tail()" # 5 ultimos datos.

Atributos, funciones y metodos

Los atributos son las caracteristicas de una base de datos o sus propiedades estructurales: que tipo de datos contiene, sus dimensiones, etc. Las funciones y metodos son algoritmos que podemos crear o manipular para realizar un analisis de datos efectivo.

 
						# Atributos
						"shape" # Dimensiones del dataframe
						"index" # Comportamiento del dataframe
						"columns" # obtener el nombre de las columnas 
						"dtype" #obtener el tipo de datos de las variables

						# metodos
						"info()" #resumen del dataframe
						"describe()" #summary del dataframe

						#Funciones
						"len" #numero de filas
						"max" #maximo index
						"min" #minimo index
						"type" #tipo de data
						"round" #redondear valores del dataframe
